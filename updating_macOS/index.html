Updating macOS

Keeping an up-to-date system is very easy using a few tools to update your system and apps in bulk.

Latest

Latest is a free and open source app for macOS that checks if all your apps are up to date. Get a quick overview of which apps changed and what changed and update them right away. Latest currently supports apps downloaded from the Mac App Store and apps that use Sparkle for updates, which covers most of the apps on the market.

Open Latest to get an idea of how up-to-date your system is. While you can update through Latest and the GUI of each out-of-date app, it’s currently easier to apply updates in bulk using the methods below and then come back to Latest to see if anything could not be updated automatically.

Mac App Store (MAS)

The simplest way to update MAS apps is through the GUI, navigating to the Updates tab, and selecting ‘Update All’.

Homebrew

Homebrew is an application for managing apps through your CLI (Command Line Interface). It makes it very simple to install, update and uninstall many CLI apps.
Homebrew Cask is an extension of Homebrew which extends the same functionality to GUI apps.

Installation

Installation for Homebrew is quite easy, and Cask is included by default. This is done from the command line; Terminal or iTerm are great choices.

Install brew: /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Basic Syntax

Function Command
update the database
update all out-of-date apps brew upgrade1
search for an app brew search
cleanup old files brew cleanup
install brew package brew install name-of-app
updates all out-of-date casks2 brew cask upgrade1
install cask app brew cask install name-of-app

Update Everything

The easiest way to quickly update your system with brew is to combine commands with the && operator. You can add everything you want to do in one line: brew upgrade && brew cask upgrade && brew cleanup

Simplied Checklist

  1. ‘Update All’ in Mac App Store
  2. Run brew upgrade && brew cask upgrade && brew cleanup
  3. Check Latest for straglers
  4. Run brew cask upgrade name-of-stragler
  5. Check Latest again; update any remaining apps manually

  1. upgrade commands will automatically trigger a brew update prior to upgrading, so you can skip brew update when updating apps. ↩↩

  2. This only updates apps that do not have the “auto-update” tag. This usually includes all apps that have an in-app updater. To update those apps from brew you can use brew cask update name-of-app. ↩
